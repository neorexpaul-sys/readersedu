# ReadersEdu App

AI-powered math thinking platform for kids aged 8-14.

> "정답을 맞추는 곳이 아니라 생각하는 힘을 키우는 공간"
> — A space to grow thinking power, not just get right answers.

## What is ReadersEdu?

ReadersEdu transforms math learning into an adventure. Kids discover their thinking style, hatch a companion pet, and grow a "thought garden" — all while building real problem-solving skills.

### Core Features (MVP)

- **Thinking Style Discovery** — 3-question diagnostic to identify logical, intuitive, creative, persistent, or collaborative thinking styles
- **Companion Pet** — Hatch and evolve a pet (Egg → Baby → Adult → Legendary) through learning activities
- **Thought Garden** — Plant mathematical concept "seeds" that grow as you practice
- **Ability Stats** — 6 dimensions: Analysis, Intuition, Persistence, Creativity, Explanation, Resilience
- **TTS Voice** — Korean text-to-speech feedback and encouragement

## Design System

- **Theme:** Deep Space (`#0a0a1a`), Amber (`#c9a84c`)
- **Dark mode, mobile-first** (375×812)
- **Korean primary, English global-ready**

## Tech

- Pure HTML/CSS/JS — single-file prototypes, no build step
- Web Speech API for TTS
- LocalStorage for data persistence
- PWA-ready meta tags
