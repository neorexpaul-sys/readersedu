readersedu.app Global B2C Architecture

UX Flow · Gamification Loop · Monetization — English-first, i18n-ready, 8–14 age target

Key Pivot Points

  • ❌ 3-question diagnosis → ✅ First math challenge → pet hatches
  • ❌ Korean B2B2C → ✅ Global standalone B2C app
  • ❌ Fear of failure → ✅ Wrong answers = garden fertilizer
  • ❌ Text-heavy UI → ✅ Visual-first, icon-driven design
  • ❌ Single-language → ✅ English default + 15 i18n locales

Retention Loop

  • Daily: 3-problem challenge (5 min) + streak + pet feeding
  • Weekly: Global cooperative quest + detective case
  • Ongoing: Pet evolution, garden growth, ability progression
  • Social (opt-in): Duo mode, garden visits, weekly showcase
  • Recovery: 3-day streak grace period, return bonuses

Monetization Ethos

  • ✅ No Pay-to-Win: All math progress is skill-based
  • ✅ No Ads: Free tier is truly ad-free
  • 💎 Cosmetics: Pet skins, garden decor, accessories
  • ⚡ Convenience: More challenges, faster energy, extended hints
  • 🎁 Seasonal: Limited-edition pets & items (FOMO)
